# 网球训练计划

## 一句话

面向网球爱好者的系统化训练计划小程序，覆盖球员画像、本周训练、训练库、NTRP 进阶、训练记录、复盘修正和成长数据。

## 解决的问题

普通球友最缺的是持续训练结构：今天练什么、练几组、怎么算有效、练完怎么调整。这个项目承担网球 AI 产品体系里的“计划大脑”角色。

## 核心功能

- 新手引导：水平、目标、频率、时长、训练条件、比赛问题画像。
- 训练库：基础训练、自选组课、NTRP 进阶、发球机模式、训练组合。
- 本周训练：今日训练、等级、经验值、训练摘要。
- 数据页：专项成长、水平进阶、阶段复测、复盘修正建议。
- 课程蓝图：40+ 训练内容，带模板质量和覆盖度校验。

## 技术栈

- 原生微信小程序
- JavaScript 模块化工具函数
- 本地数据与训练模板
- Node 脚本验证训练流程、模板质量和蓝图覆盖

## AI Builder 亮点

- 已经具备 Agent 所需的上下文：球员画像、目标、训练条件、历史记录和训练库。
- 可以作为主项目训练处方的下游：视频识别发现弱项后，把建议转成训练计划。
- 展示你能做长期产品闭环，而不是只做一次性的 AI Demo。

## 商业化方向

- 给网球俱乐部做训练计划工具。
- 给教练做课前/课后训练安排助手。
- 给硬件或相机产品补充软件服务层。

## Demo 讲法

展示从新手画像到本周训练，再到训练详情和数据页。强调它是“低成本训练教练系统”，不是训练内容库。
