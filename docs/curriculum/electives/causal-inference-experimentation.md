# XCA · 因果推断与实验

- **建议时长：**6 周 · 每周 12-15 小时
- **先修课程：**AI01、ML01
- **课程性质：**兴趣选修
- **验收方式：**从零构建 + 因果分析项目 + 口试

## 方向目标

从相关性预测走向干预和决策，学习因果图、混杂、实验、准实验和异质效应，避免把模型相关性错误包装成业务因果。

## 周计划

| 周 | 主题与实践 |
| --- | --- |
| 1 | 潜在结果、反事实、ATE 与识别问题 |
| 2 | DAG、d-separation、混杂、碰撞点和选择偏差 |
| 3 | 随机实验、功效、样本量、分层和序贯风险 |
| 4 | 倾向评分、匹配、加权和双重稳健估计 |
| 5 | 差分中的差分、断点、工具变量和假设检查 |
| 6 | 异质效应、政策评估与业务实验报告 |

## 从零构建

[进入逐步构建指南](../../builds/causal-experiment-lab.md)

实现随机化模拟器、均值差估计、倾向评分加权和 bootstrap 置信区间，用合成数据演示混杂与碰撞点偏差。

## 真实交付

为一个产品或运营决策设计实验或准实验，提交识别假设、样本量、指标、停止规则、风险和决策阈值。

## 主要资源

- [Causal Inference: The Mixtape](https://mixtape.scunning.com/)
- [Causal Inference for the Brave and True](https://matheusfacure.github.io/python-causality-handbook/)
- [DoWhy 文档](https://www.pywhy.org/dowhy/)

## 必交证据

- 合成数据真值、估计值与置信区间对照
- 至少一个混杂、选择偏差或错误控制变量反例
- 实验/准实验假设、功效和决策报告

## 反 bypass 口试

1. 预测准确为什么不能证明干预有效？
2. 控制更多变量为什么可能引入偏差？
3. A/B 测试显著但不应上线的情况有哪些？
