# 网球记分复盘

## 一句话

原生微信小程序，用于业余网球比赛的快速记分、技术数据记录、赛后复盘和训练建议生成。

## 解决的问题

比赛结束后，很多球友只记得输赢，不知道问题来自发球、一发率、非受迫失误、关键分还是打法选择。这个项目把比赛过程沉淀成后续训练的输入。

## 核心功能

- 创建单打 / 双打比赛，支持一盘制、三盘两胜、抢七、抢十。
- 支持 No-Ad 和 Advantage。
- 横屏计分板和纵屏详细记录。
- 每分记录 Ace、双误、一发/二发、制胜分、受迫失误、非受迫失误。
- 赛后复盘展示比分、关键数据、发球统计、优势项、问题项和训练建议。
- 已有计分规则测试，覆盖 Deuce、金球、抢七、抢十、撤销和趋势。

## 技术栈

- 原生微信小程序
- 本地离线存储，预留微信云开发同步
- Node 测试脚本验证计分规则

## AI Builder 亮点

- 这是 Agent 的问题来源：比赛复盘发现短板，再交给训练计划和视频识别系统处理。
- 规则严谨度高，适合展示你能处理真实业务规则，而不是只做聊天界面。
- 当前仓库有未提交改动，短期作为能力展示，不作为第一阶段主入口。

## 商业化方向

- 球局活动记分和成绩认领。
- 俱乐部赛事数据沉淀。
- 与训练计划联动，形成“比赛问题 -> 训练任务”的闭环。
