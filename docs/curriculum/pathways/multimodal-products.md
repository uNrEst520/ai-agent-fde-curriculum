# 多模态产品路线

## 适合目标

计算机视觉、语音、视频、生成媒体、智能硬件或人机交互方向的 AI 产品与工程岗位。

## 推荐顺序

```text
O00 → SE01 → SE02 + SE03 + SE04
→ AI01 → ML01 → DL01
→ AP01 → EV01
→ XCV +（XSP 或 XGM）
→ AP03 → AG01 → PR01
→ FD01 → XDS / FD02
```

涉及视频理解优先 XCV，实时助手优先 XSP，内容生产优先 XGM，机器人与环境决策再增加 XRL。

## 作品集要求

- 一个从底层信号或像素到模型输出的理解项目
- 一个包含时序、多模态输入或实时交互的产品
- 分设备、环境、用户和失败类型的评测报告
- 隐私、授权、素材来源和人工确认设计

## 竞争力

能够把感知模型、生成模型、交互和真实设备/环境连接起来，并用系统级指标而非单一模型分数验收体验。
