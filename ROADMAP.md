# 课程建设路线图

## v1.0：长期课程骨架

- README 直接展示约 108 周主线、先修关系和兴趣选修。
- 为每门主线课和选修课建立详情页。
- 建立四条目标路线与 Build to Understand 双项目制。
- 用结构化目录自动检查课程编号、详情页、README 入口和先修环。

## v1.1：逐课教学包

- 为 O00、SE01、SE02、AI01、ML01 优先补齐逐日课程。
- 每周配置视频/原始资料、练习、记忆清单、评分表和口试。
- 发布挑战测试，让有经验的学习者用等价证据跳过已掌握课程。
- 建立公开、脱敏的优秀作业样例。

## v1.2：模型与应用实验室

- 发布自动微分、微型 Transformer、RAG、MCP 和 Agent runtime 的参考测试。
- 加入可重复实验环境、固定数据集和性能基线。
- 建立从研究雷达到正式课程的复现队列。

## v1.3：FDE 与领域工作室

- 发布业务访谈、机会评分、试点、ROI 和采用工具包。
- 建立体育、建筑、咨询等脱敏领域案例。
- 完成第一批真实或高度仿真的企业 AI 上岗毕业项目。

## 持续维护

每季度检查资源、工具、规范与工程实践；每年复审稳定主线；研究雷达每周收集候选，但论文必须经过证据审查、最小复现和课程 RFC 才能成为正式内容。
