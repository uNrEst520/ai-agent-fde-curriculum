# XSP · 语音与音频 AI

- **建议时长：**6 周 · 每周 12-15 小时
- **先修课程：**DL01
- **课程性质：**兴趣选修
- **验收方式：**从零构建 + 方向项目 + 口试

## 方向目标

学习波形、频谱、语音识别、语音合成、说话人和实时交互，理解延迟、噪声与轮次管理如何决定体验。

## 周计划

| 周 | 主题与实践 |
| --- | --- |
| 1 | 波形、采样、傅里叶变换与声谱图 |
| 2 | 音频特征、分类与数据增强 |
| 3 | 自动语音识别、对齐和错误率 |
| 4 | 语音合成、音色、韵律和安全 |
| 5 | 端点检测、打断、流式处理和对话状态 |
| 6 | 实时语音助手与噪声场景评测 |

## 从零构建

[进入逐步构建指南](../../builds/realtime-voice-assistant.md)

实现短时傅里叶变换和简化音频分类器，再搭建可打断、可恢复、记录延迟的语音处理流水线。

## 真实交付

交付一个真实语音任务，在不同设备、噪声、口音和网络条件下评测准确率、首响应和端到端延迟。

## 必交证据

- 可运行代码与可复现实验环境
- 基线、切片指标、失败样本和风险记录
- 面向用户或业务负责人的演示与价值说明

## 反 bypass 口试

1. 字错率低为什么不等于任务完成率高？
2. 流式语音系统的延迟来自哪些环节？
3. 声音克隆产品必须增加哪些同意与滥用控制？
