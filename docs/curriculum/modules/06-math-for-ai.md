# AI01 · AI 数学基础

- **建议时长：**8 周 · 每周 12-15 小时
- **先修课程：**SE01
- **课程性质：**主线课
- **验收方式：**构建项目 + 交付任务 + 口试

## 为什么学

数学的价值不是手算复杂公式，而是提供描述变化、相似性、不确定性和优化的语言，让你能解释模型行为并设计实验。

## 学习目标

- 用向量、矩阵、导数和概率解释常见模型
- 从损失函数推导简单梯度更新
- 识别统计波动、数据泄漏和错误因果推断

## 周计划

| 周 | 主题与实践 |
| --- | --- |
| 1 | 函数、指数、对数与数量级 |
| 2 | 向量、矩阵、线性变换与几何 |
| 3 | 特征值、分解、相似度与降维直觉 |
| 4 | 微分、偏导、链式法则与梯度 |
| 5 | 优化、学习率、凸性与正则化 |
| 6 | 概率、条件概率、贝叶斯与随机变量 |
| 7 | 期望、方差、分布、抽样与置信区间 |
| 8 | 假设检验、相关与因果边界；综合实验 |

## 从零构建

使用 Python 但不调用机器学习库，构建可视化实验室展示矩阵变换、梯度下降、抽样误差和贝叶斯更新。

## 真实交付

选择一个真实业务指标，完成数据生成过程、统计假设、误差来源和决策风险报告。

资源用于建立主线，最新论文与工具变化从 [研究雷达](../../../research/README.md) 进入候选，不会未经验证直接替换稳定原理。

## 必交证据

- 至少 12 个可视化数学实验
- 一次手推结果与代码结果对照
- 一份错误统计结论的反例分析

## 反 bypass 口试

1. 余弦相似度忽略了什么信息？
2. 链式法则为何是深度学习的核心？
3. 统计显著为什么不等于业务重要？
