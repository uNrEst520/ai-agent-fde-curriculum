# XGM · 生成式媒体与扩散模型

- **建议时长：**7 周 · 每周 12-15 小时
- **先修课程：**DL01
- **课程性质：**兴趣选修
- **验收方式：**从零构建 + 方向项目 + 口试

## 方向目标

从概率生成、自动编码器和扩散过程出发，学习图像/视频生成、条件控制、工作流设计、质量评测与版权安全。

## 周计划

| 周 | 主题与实践 |
| --- | --- |
| 1 | 生成建模、潜变量、VAE 与表示空间 |
| 2 | 噪声过程、去噪目标和采样 |
| 3 | 实现微型扩散模型 |
| 4 | 条件生成、文本编码与 classifier-free guidance |
| 5 | LoRA、控制条件、编辑与一致性 |
| 6 | 视频生成、时序一致性和工作流工程 |
| 7 | 质量、多样性、安全、来源与商业交付 |

## 从零构建

在小型数据集上实现前向加噪、去噪网络、训练目标和采样循环，解释每一步的概率与工程作用。

## 真实交付

构建可重复的生成媒体工作流，包含版本、种子、素材来源、质量评审、失败样本和安全检查。
