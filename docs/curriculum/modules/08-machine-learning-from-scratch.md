# ML01 · 机器学习从零实现

- **建议时长：**7 周 · 每周 12-15 小时
- **先修课程：**AI01、DE01
- **课程性质：**主线课
- **验收方式：**构建项目 + 交付任务 + 口试

## 为什么学

机器学习是一套从数据中估计规律并管理泛化误差的方法。掌握经典 ML，才能理解评测、特征、数据泄漏和深度学习并非魔法。

## 学习目标

- 建立训练、验证、测试与基线的正确实验流程
- 从零实现主要监督和无监督算法
- 根据错误类型、数据量和业务代价选择指标与模型

## 周计划

| 周 | 主题与实践 |
| --- | --- |
| 1 | 问题定义、特征、标签、基线与数据切分 |
| 2 | 线性/逻辑回归、损失、优化与正则化 |
| 3 | 决策树、集成学习与特征重要性 |
| 4 | 距离、kNN、聚类与降维 |
| 5 | 概率分类、校准与不确定性 |
| 6 | 评测、交叉验证、泄漏、偏差与公平性 |
| 7 | 从零工具箱与成熟库对照项目 |

## 从零构建

[进入逐步构建指南](../../builds/tiny-ml-library.md)

仅用 NumPy 实现线性回归、逻辑回归、决策树、k-means、数据切分和常用指标。

## 真实交付

用成熟库完成一个端到端 ML 项目，包含数据卡、基线、误差分析、模型卡和可复现实验。

## 必交证据

- 自实现与成熟库结果对照
- 至少一次数据泄漏实验
- 按业务代价选择指标的说明

## 反 bypass 口试

1. 测试集为什么不能参与模型选择？
2. 准确率很高的模型为何可能毫无价值？
3. 正则化改变了什么偏好？
