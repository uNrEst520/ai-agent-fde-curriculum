# AI Builder / FDE 路线

## 适合目标

AI 应用工程师、Agent 工程师、AI 产品工程师、工作流工程师、FDE、企业 AI 顾问或一人公司。

## 推荐顺序

```text
O00
→ SE01 → SE02 + SE03 → DE01
→ AI01 → ML01
→ AP01 → AP02 + AP03 → AG01 → EV01
→ FD01（可在 AP01 后并行）
→ PR01 → FD02
→ XDS 或一个行业相关选修；高风险企业项目增加 XGV
```

SE04、DL01 和 LM01 不建议永久跳过。转岗路线可以在第一个 FDE 项目后补完，以缩短首次作品集时间。

## 作品集要求

- 一个有用户流程、数据、评测和部署的 AI 应用
- 一个自建 Agent runtime 或工具/MCP 项目
- 一个与行业匹配的 XDS、XKG、XCA 或 XGV 方向项目
- 一次企业发现、试点、培训、验收和价值复盘
- 一份能够解释失败边界与人工接管的风险报告

## 竞争力

这条路线的优势不是“会调用更多模型”，而是能在企业现场把模糊问题转成可控流程，并跨业务、产品、模型和工程推动结果。
