# AI02 · 经典 AI：搜索、规划与知识表示

- **建议时长：**5 周 · 每周 12-15 小时
- **先修课程：**SE01
- **课程性质：**主线课
- **验收方式：**构建项目 + 交付任务 + 口试

## 为什么学

Agent 并不是从大语言模型开始的。搜索、约束、规划、博弈与知识表示提供了可解释的决策基础，也帮助我们判断何时不需要 LLM。

## 学习目标

- 把问题表达为状态、动作、目标、代价与约束
- 实现并比较无信息搜索、启发式搜索和博弈搜索
- 理解符号规划与概率模型相对 LLM 的优势边界

## 周计划

| 周 | 主题与实践 |
| --- | --- |
| 1 | 问题形式化、BFS、DFS 与代价搜索 |
| 2 | 启发式、A*、局部搜索与搜索质量 |
| 3 | 约束满足、回溯、传播与调度 |
| 4 | 博弈树、minimax、剪枝与不确定决策 |
| 5 | 逻辑、知识表示、规划与混合智能体 |

## 从零构建

从零实现路径搜索器、课程排程约束求解器和一个带 minimax 的小游戏智能体。

## 真实交付

选择一个业务流程，把规则、搜索和 LLM 三种方案做成对照实验，说明正确性、成本和可维护性。

## 必交证据

- 三个算法的性能与正确性对照
- 一个不可行约束案例
- 规则、算法与 LLM 的选型记录

## 反 bypass 口试

1. A* 的启发式函数需要满足什么性质？
2. 什么时候规划问题不应交给 LLM？
3. 状态表达方式如何决定搜索难度？
