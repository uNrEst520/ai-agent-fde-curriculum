# Curriculum Index

课程内容按稳定性分为四层：

| 层级 | 更新频率 | 内容 |
| --- | --- | --- |
| 稳定核心 | 每年复审 | 计算思维、Web/API、数据、Agent 基本模型、评测、安全、交付 |
| 实践模块 | 每季度复审 | RAG、MCP、Coding Agent、可观测性、部署与行业方案 |
| 工具实验 | 每月复审 | 当前模型、SDK、平台和开发工具 |
| 研究雷达 | 每周生成 | 新论文、基准、协议与官方工程实践候选 |

正式课程入口见 [docs/curriculum/README.md](docs/curriculum/README.md)。
