# 变更记录

课程版本采用语义化思想：

- `MAJOR`：培养目标、核心能力或毕业标准发生变化。
- `MINOR`：替换课程、增加专修或新增项目，但能力目标不变。
- `PATCH`：修复链接、表述、环境说明和非实质性问题。

## 1.1.0 - 2026-07-17

- 新增 AI 全领域覆盖图，并补充因果推断、知识图谱、机器人和 AI 治理选修。
- 为全部 31 门课程建立一一对应的 Build Your Own AI 逐步构建指南。
- README 新增可视化知识地图、旗舰构建项目和方向实验室入口。
- 课程目录新增 build_page，自动校验课程、构建页、项目目录和双向链接。
- 新增构建项目自动测试，要求每个项目至少六个“构建-验证-解释”关卡。

## 1.0.0 - 2026-07-17

- 将课程从 12 周 Agent/FDE 计划重构为约 108 周的完整 AI 主线。
- README 直接提供可点击课程地图、先修关系、时长和构建项目。
- 新增 19 门主线课程、8 门兴趣选修和 4 条目标路线的详情页。
- 引入 Build to Understand 双项目制：从零实现用于理解，成熟工具用于交付。
- 新增结构化课程目录及编号、页面、README 入口、先修环和总时长检查。
- 旧版 12 周与 3 个月计划转为历史加速路线。

## 0.1.0 - 2026-07-17

- 将原 3 个月转型计划升级为长期 AI Agent / FDE 开放课程。
- 新增能力模型、12 周共同核心、终身进阶路线和 FDE 毕业项目。
- 新增论文雷达、研究转化流程和自动更新工作流。
- 新增课程验证脚本、GitHub Issue 模板和贡献规则。
