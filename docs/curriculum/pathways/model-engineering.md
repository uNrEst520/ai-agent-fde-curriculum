# 模型与算法工程路线

## 适合目标

机器学习工程师、深度学习工程师、LLM 工程师、训练/推理系统工程师，以及希望深入理解模型而非只构建应用的学习者。

## 推荐顺序

```text
O00 → SE01 → SE03 + SE04 + DE01
→ AI01 + AI02 → ML01 → DL01 → LM01
→ XLM + XRE +（XCA 或 XKG）
→ EV01 → PR01
```

计算机视觉、语音、生成媒体、强化学习、因果推断、知识图谱或推荐系统中至少选择一门，形成算法方向深度。

## 作品集要求

- 一套从零实现的 ML/DL 组件及梯度、正确性测试
- 一个微型语言模型或方向模型的训练报告
- 自实现与成熟框架的性能、正确性和维护成本对照
- 一次论文复现或系统优化实验

## 竞争力

能够从数据、目标函数、模型、训练、推理和评测解释系统行为，并知道应用问题何时来自模型、数据或系统约束。
