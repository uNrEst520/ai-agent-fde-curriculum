# XCV · 计算机视觉与多模态

- **建议时长：**8 周 · 每周 12-15 小时
- **先修课程：**DL01
- **课程性质：**兴趣选修
- **验收方式：**从零构建 + 方向项目 + 口试

## 方向目标

从像素、卷积和视觉表示出发，进入检测、姿态、视频与视觉语言模型，最终构建可评测的多模态产品。

## 周计划

| 周 | 主题与实践 |
| --- | --- |
| 1 | 图像表示、颜色、采样、增强与数据标注 |
| 2 | 卷积、特征、分类与迁移学习 |
| 3 | 目标检测、分割、关键点与姿态 |
| 4 | 视频、跟踪、时序与动作识别 |
| 5 | 视觉 Transformer、对比学习与多模态表示 |
| 6 | 视觉语言模型、提示、工具与结构化输出 |
| 7 | 数据偏差、鲁棒性、隐私与多模态评测 |
| 8 | 完整视觉/视频产品与用户验证 |

## 从零构建

用 NumPy/PyTorch 基础算子实现卷积、边缘检测和小型分类器；再构建一条含检测或姿态、时序分析和错误可视化的视频流水线。

## 真实交付

交付一个真实视觉任务，报告数据来源、标注质量、分场景指标、速度、失败画廊和人工接管策略。
